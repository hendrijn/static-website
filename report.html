  1. The website is for WIT club to create a central location to look at information for members.
  2. All text is my own, with images provided by open licence google searches.
  3. Home, Exec, Events, Fundraisers, Contact Us. Home is the .index file.
  4. The header contains the WIT logo and the navigation tabs.
  5. The nav links are placed in the header and link to each file.
  6. I used media queries to change the font size for header/footer, adjust the events and fundraisers page to be monocolumn, and create the hamburger menu when viewing on mobile screens.
  7. I created a main.css file which houses all formatting for the header and footer. Since events and fundraising had the same styling, I created a joint css file for those too. All other page styling was in a separate file.
  8. All images have an alt attribute that matches the content of the image.
  9. The favicon for each page is a smaller version of a WIT logo that appears in the tab.
  10. All images are in the images folder, all css files are in a css folder, and all js files are in a js folder.
